Money in the Bank was my Phase 3 project at DigitalCrafts. The core learning objective for this project was to create a web application using React.

This was my first solo project, and I really enjoyed learning React while creating this web app. The concept is simple: enter your recurring income/expenses on the Transactions tab, then see how your budget affects your bank balance over time on the other tabs.

Significant future development is planned for this app.

BankHubclose

BankHub was my Phase 2 group project at DigitalCrafts. The core learning objective for this project was to create a web application with primary focus on backend technologies such as Node.js, Express, Knex, and PostgreSQL.

My primary role consisted of setting up the database (schema, migrations, and seeding), querying the database for CRUD operations related to transferring data between users, and rendering information from the database on the user account page.

Rendering Returnsclose

Rendering Returns was my Phase 1 group project at DigitalCrafts. The core learning objective for this project was to utilize JavaScript libraries and external APIs.

My primary role consisted of implementing the Alpha Vantage API for the stock data, and the stock ticker. One challenge I encountered was handling the API call limit for Alpha Vantage. To keep the site usable when the API call failed, I imported historical stock data to a JSON file and accessed that in place of the API response.
